aruba

CLI Steps for Cucumber

Install

# Gemfile
gem 'aruba', '0.2.0'

# features/support/aruba.rb
require 'aruba'

For Ruby executables

When I run "my_ruby_executable"
Then I the output should match /zomg ruby 1.8.7/

RVM

Given I am using rvm "ruby-1.9.1"
  And I am using an empty rvm gemset "test"
 When I run "my_ruby_executable"
 Then I the output should match /zomg ruby 1.9.1/

Output

When I run "curl www.bostonrb.org"
Then the stdout should contain "Synchronizing Core Data"
 And the stderr should contain ""

When I run "curl fail.bostonrb.org"
Then the stderr should contain "Couldn't resolve host"
 And the stdout should contain ""

Files

When I run "curl www.bostonrb.org -o test.output"
 And the following files should exist:
     | test.output |
Then the file "test.output" should contain "Synchronizing Core Data"

Exit status

When I run "curl www.bostonrb.org"
Then the exit status should be 0

When I run "curl fail.bostonrb.org"
Then the exit status should not be 0
 And the exit status should be 6
